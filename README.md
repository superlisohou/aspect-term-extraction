# aspect-term-extraction
Required package:

    pandas 0.19.0,
    
    numpy 1.14.1,
    
    sklearner 0.0,
    
    ntlk 3.2.5,
    
    CRF++ 0.58,
    
To perform logistic regression: python3.5 main.py --a lr

To perform conditional random field: python3.5 main.py --a crf --p "path of CRF++-0.58 fold"
